# Community and Discussions

GitHub Discussions is the preferred place for questions, ideas, project updates, and examples related to this work-in-progress benchmark.

## Recommended categories

| Category | Purpose | Format |
|---|---|---|
| 📣 Announcements | Updates and news from the project maintainer | Announcement |
| #️⃣ General | Anything relevant to the project that does not fit another category | Open-ended discussion |
| 💡 Ideas | Suggestions for changing or improving the benchmark | Open-ended discussion |
| 🗳️ Polls | Multiple-choice questions for community feedback | Poll |
| 🙏 Q&A | Technical or usage questions that can have an accepted answer | Question and Answer |
| 🙌 Show and tell | Experiments, extensions, plots, or tests based on the project | Open-ended discussion |

## What belongs in Discussions

Use Discussions for:

- questions about numerical methods, setup, or interpretation
- ideas for new solver variants or benchmark cases
- examples built from the repository
- requests for clarification
- community polls
- project announcements

## What belongs in Issues

Use Issues for work that has a clear, actionable outcome:

- reproducible bugs
- incorrect documentation
- failed build or run commands
- missing files
- specific implementation tasks
- tracked benchmark improvements

## Before posting

Please search existing discussions and issues first. Include the implementation, operating system, compiler or interpreter version, run command, parameter values, and relevant logs whenever they affect the question.

## Project status

The repository is still under active development. Runtime tables and numerical conclusions may change as convergence reporting, repeated measurements, and benchmark metadata are improved.
